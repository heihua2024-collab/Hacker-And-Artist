# splat-compression

## 术语

- **中文**：Splat 压缩
- **English**：Splat compression

## 分层

- **short**：把几百万颗高斯的参数打包得更紧，像把羽绒服从行李箱里抽真空，方便网上传、手机载。
- **定义**：在保持可接受视觉质量下量化、熵编码或结构化稀疏化 splat 属性；SPZ、KSplat 等格式属于工程实践。
