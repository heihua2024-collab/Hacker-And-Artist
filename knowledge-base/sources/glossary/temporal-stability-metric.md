# temporal-stability-metric

## 术语

- **中文**：时序稳定性指标
- **English**：Temporal stability metric

## 分层

- **short**：看视频或连续视角时，画面不应一会儿跳色一会儿抖影；时序指标就是在量「相邻帧之间变化是不是平滑」。
- **definition**：动态神经渲染中可报告帧间 LPIPS、光流一致性或人工评分；3DGS 视频扩展工作常显式优化时序项。
