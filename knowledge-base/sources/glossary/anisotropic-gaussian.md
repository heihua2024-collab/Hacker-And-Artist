# anisotropic-gaussian

## 术语

- **中文**：各向异性高斯
- **English**：Anisotropic Gaussian

## 分层

- **short**：像一滴颜料可以被拉成长条或压成扁片，而不是永远正圆；方向不同，扩散形状就不同。
- **definition**：各向异性高斯的三维协方差一般非对角单位缩放，对应椭球等概率面。3DGS 用此类高斯表达表面局部几何，配合可微光栅化实现视角相关外观。
