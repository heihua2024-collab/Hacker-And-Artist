# Gracia（keep slug: `gracia-3d-flow`）

> Opus keep 列表使用 slug `gracia-3d-flow` 指代 **Gracia** 平台的端到端 **4D Gaussian Splatting（4DGS）** 体积视频生产与分发能力；公开品牌名为 **Gracia**，无单独名为「Gracia 3D Flow」的 SKU，本条目以 **Gracia** 为准。

## 一句话

基于 4DGS 的体积视频基础设施：云端处理、VR / WebXR / 2D 流式播放，面向影视广告与 XR 活动。

## 客观功能（可验证）

- 官网与 Radiance Fields 资料均描述：**端到端** 4DGS 内容生产与部署，覆盖美国、欧盟、英国、加拿大、日本等地的拍摄合作与云端处理。
- **WebXR 流式**：官网写明 4DGS 内容可在 WebXR 中流式播放，无需下载应用（受带宽与设备条件约束）。
- **SDK**：存在公开 GitHub 仓库 `gracia-labs/web-sdk`，用于 Web / 引擎侧集成（仓库自述为 Gracia Web SDK）。
- **引擎插件**：官网列出 Unity、Unreal、Blender、Cinema 4D、Adobe AE 等插件状态多为 **in development**（以官网当前文案为准）。
