# Remy（kiri-remy）

## 摘要

- **slug**: `kiri-remy`
- **厂商**: 麒砺创新技术（深圳）有限公司（对外品牌 KIRI Innovations）
- **类别**: capture（视频生成可查看的三维记录；与 KIRI Engine 同属一家公司）
- **主条目**: 参见 Track A 的 `kiri-engine.md`（若已存在）；本条记录其子产品 Remy。

## 客观描述

搜狐号转载稿介绍麒砺创新在华为开发者大会展示 Remy，并描述其通过拍摄视频生成三维空间记录、结合鸿蒙能力在手机端查看等要点。Remy 与 KIRI Engine 的定位差异在同文中被概括为从效率工具走向大众记录产品。

## Verbatim 证据

> Remy是一款通过拍摄视频生成逼真3D空间的全新记录工具。它巧妙地将KIRI多年的技术积累与华为鸿蒙系统强大的3D图像能力相融合，在手机端实现了3D影像的查看，极大地降低3D创作的成本和门槛。
