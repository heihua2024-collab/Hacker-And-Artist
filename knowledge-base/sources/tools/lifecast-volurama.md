# Volurama（Lifecast）

## 摘要

Volurama 是 Lifecast 公司的 Windows / macOS 图形界面，使用 iPhone 视频输入驱动 NeRF 引擎重建静态场景。GitHub 上存在 lifecast_public 相关开源组件。

## 厂商与定位

- **厂商**：Lifecast Inc.
- **官网**：https://volurama.com/
- **阶段 1 分类**：training
- **区域**：north-america

## Verbatim 证据（阶段 1）

> Lifecast's main product is Volurama - a Windows/Mac GUI for the NeRF engine for reconstructing static scenes from iPhone video input.
