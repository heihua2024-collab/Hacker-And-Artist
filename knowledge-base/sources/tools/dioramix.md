# Dioramix

## 摘要

Dioramix 是由 Olli Huttunen 开发的基于浏览器的编辑器，使用 Babylon.js 与 WebGPU，将 PLY、SPZ、SOG 与传统网格组织为可演示的叙事。Radiance Fields 2026-04-28 指出公开 beta 尚未完全开放，sandbox 需向 dioramix.eu 申请。

- **官网**：https://dioramix.eu/
