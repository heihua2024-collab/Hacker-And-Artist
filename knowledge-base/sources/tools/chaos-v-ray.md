# Chaos V-Ray

## 摘要

Chaos V-Ray 在 3ds Max、SketchUp、Revit、Cinema 4D 等 DCC 中提供高保真渲染；Radiance Fields 说明 V-Ray 7 公测支持导入与渲染 Gaussian Splatting 训练的 PLY。

- **区域**：europe
