# Portality

## 摘要

Portality 提供基于 WebGL2 的浏览器端 Gaussian Splatting 渲染实现，Radiance Fields 说明其使用 GPU bitonic sort。公开 viewer 域名 viewer.portality.ai。

## 厂商与定位

- **厂商**：Portality
- **官网**：https://portality.ai/
- **阶段 1 分类**：viewing
- **区域**：global

## Verbatim 证据（阶段 1）

> Portality is a WebGL2 implementation of a browser-based renderer for Gaussian Splatting.

- **来源**：https://radiancefields.com/platforms/portality
