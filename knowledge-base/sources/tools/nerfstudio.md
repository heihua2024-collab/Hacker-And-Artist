# nerfstudio

## 摘要

nerfstudio 是由 UC Berkeley 相关团队维护的开源 Radiance Field 研究管线，集成多种 NeRF 与 3DGS（含 Splatfacto）方法，并提供 Viser 查看器。文档站位于 docs.nerf.studio。

## 厂商与定位

- **厂商**：nerfstudio project / BAIR collaborators
- **文档**：https://docs.nerf.studio/

- **区域**：north-america
