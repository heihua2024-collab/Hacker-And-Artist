# COLMAP

## 摘要

COLMAP 是广泛使用的开源 Structure-from-Motion 与多视图立体视觉软件，为高斯泼溅与 NeRF 管线提供相机位姿与稀疏重建输入。代码与发布在 GitHub colmap/colmap。
