## 工作流速记

在依赖中安装 `three`（peer `>=0.160.0`）与本库，使用其 Viewer / `addSplatScene` 加载 `.ply`、`.splat` 或 `.ksplat` 于浏览器中实时渲染；这是独立 npm 包，不是 `three` 默认发行版的一部分。导出压缩 splat 可走 README 所述转 `.ksplat` 流程或 `KSplatLoader.downloadFile`。
