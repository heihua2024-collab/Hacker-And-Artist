# LangSplat（2312.16084）

- **论文标题**：LangSplat: 3D Language Gaussian Splatting
- **会议**：CVPR 2024
- **arXiv**：https://arxiv.org/abs/2312.16084
- **项目页**：https://langsplat.github.io/
- **代码**：https://github.com/Minghanqin/LangSplat
- **摘要首句**：arXiv Atom `summary` 第一句

## 要点

在三维高斯上蒸馏语言特征并用瓦片泼溅渲染，结合场景级语言自编码与层次语义，实现快速开放词汇三维查询。
