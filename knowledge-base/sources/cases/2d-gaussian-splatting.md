# 2D Gaussian Splatting（2403.17888）

- **论文标题**：2D Gaussian Splatting for Geometrically Accurate Radiance Fields
- **会议**：SIGGRAPH 2024
- **arXiv**：https://arxiv.org/abs/2403.17888
- **项目页**：https://surfsplatting.github.io/
- **代码**：https://github.com/hbb1/2d-gaussian-splatting

## 要点

用定向平面二维高斯盘表示薄表面，结合射线与高斯盘求交的可微光栅化与几何正则，提升网格与法线质量。
