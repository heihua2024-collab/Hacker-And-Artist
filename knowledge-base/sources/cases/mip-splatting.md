# Mip-Splatting（2311.16493）

- **论文标题**：Mip-Splatting: Alias-free 3D Gaussian Splatting
- **会议**：CVPR 2024

- **项目页**：https://niujinshuchong.github.io/mip-splatting/

- **摘要首句来源**：arXiv Atom `summary` 字段第一句（与 PDF 摘要一致）

## 要点

针对 3DGS 在改变焦距或距离时的混叠与膨胀伪影，引入三维平滑滤波与二维 Mip 滤波，实现多尺度下更稳定的渲染。
