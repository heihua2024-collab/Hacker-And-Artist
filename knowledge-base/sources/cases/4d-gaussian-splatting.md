# 4D Gaussian Splatting（2310.08528）

- **论文标题**：4D Gaussian Splatting for Real-Time Dynamic Scene Rendering
- **会议**：CVPR 2024
- **arXiv**：https://arxiv.org/abs/2310.08528
- **项目页**：https://guanjunwu.github.io/4dgs/
- **代码**：https://github.com/hustvl/4DGaussians
- **摘要首句**：arXiv Atom `summary` 第一句

## 要点

用 3D 高斯与 4D 神经体素及形变场统一表示动态场景，实现动态场景上的实时泼溅渲染。
