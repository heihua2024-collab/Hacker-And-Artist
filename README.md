# Hacker-And-Artist / TOP3DGS

黑画 | Immersive Space

INKTOYS / top3dgs.com is a Next.js site for 3D Gaussian Splatting content, tools, learning paths, cases, gallery, news, and industry insights.

## Development

```powershell
npm run dev
```

Open `http://localhost:3000`.

## Build

```powershell
npm run build
```

## Deployment

Deploy with Cloudflare Pages:

- Framework preset: `Next.js`
- Build command: `npm run build`
- Node version: `20`

Production domain: `https://top3dgs.com`
